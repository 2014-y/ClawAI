关于 AI助手

了解系统的技术架构、核心能力与极速使用指南

⚙️ 系统技术架构

AI助手是一个经过高度沙箱化封装、开箱即用的本地智能网关服务。其核心技术由以下几部分驱动：

💎 Electron 骨架层
采用无边框流光设计与高度硬件加速，完美结合了桌面端程序的极速响应与现代化 Web UI 的极致动效。
📦 内置 Node.js 沙箱
项目深度打包了 100% 独立的 Node.js 运行时及所有依赖库。用户无需安装配置任何环境，双击即可无缝启动。
🚀 OpenClaw Gateway 核心
集成 OpenClaw 本地 AI 智能体通信总线，提供极致的插件加载、多渠道协议收发与模型路由治理。

🔥 核心服务能力

通过集成本地网关，您可以赋予您的个人或团队以下高附加值人工智能服务能力：

💬 微信 AI 伴侣
扫码即可一键托管微信聊天！支持私聊回复、群聊艾特回复、图片深度看图理解以及手机语音通话直接交互。
🛡️ 本地隐私沙箱
支持通过 Ollama 跑本地离线大模型。在断网或局域网无网络环境下，100% 确保数据不出本地盘符。
🔌 40+ 生产力智能体插件
支持长期记忆存储、实时网页搜索、超长文档自动解析与摘要、自动化工作流等各种插件。

📘 极速上手使用指南

1 启动本地网关

切到“控制台”标签页，点击右侧醒目的“启动网关”大按钮。等左侧终端打印出运行成功字样，网关将开始在 18789 端口监听请求。

2 扫码绑定微信

点击控制台底部的“绑定微信”按钮，系统会自动加载并弹出一个高清登录二维码。直接使用您想要作为助手的微信扫一扫确认登录。

3 体验用量监控

切换到“用量监控”面板。此时有任何人通过微信向您的 AI 发起对话，该面板都会实时动态记录模型调用的 Tokens 指标与成本消耗。
